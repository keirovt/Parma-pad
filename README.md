# Parma-pad

Parma-pad es un sistema modular diseñado para expandir la interacción física disponible en una laptop mediante una serie de accesorios compactos que se acoplan directamente al borde inferior del teclado. El proyecto propone una solución portátil y adaptable para usuarios que requieren controles adicionales sin recurrir a periféricos externos voluminosos o dependientes de un espacio dedicado en el escritorio.

Descripción general

El sistema parte de una base estructural denominada “rail base”, una pieza delgada que sigue la geometría frontal del chasis de la laptop y permite fijar módulos adicionales sin bloquear teclas, ventilación ni puertos. Sobre esta base se acoplan distintos módulos intercambiables que amplían la experiencia de uso según las necesidades del usuario. Cada módulo está diseñado para operar de manera independiente, pero comparte un esquema estándar de comunicación y acoplamiento, lo que facilita la reorganización física y la expansión del sistema.

Arquitectura del sistema
1. Módulo base (Rail Base)

La base sirve como punto de anclaje físico para todos los módulos y define la estructura modular del proyecto. Está diseñada para soportar cargas ligeras, mantener estabilidad durante la interacción del usuario y permitir el intercambio rápido de módulos. Además del montaje mecánico, establece la ruta de comunicación y energía entre los módulos.

2. Módulo de macropad

Este módulo incluye entre 4 y 6 teclas mecánicas tipo MX, orientadas a ofrecer accesos rápidos, funciones de juego o comandos específicos configurados por el usuario. Su disposición compacta permite un uso intuitivo sin requerir espacio adicional en el escritorio. El módulo se integra a la base mediante el sistema de acoplamiento común y mantiene compatibilidad con software de asignación de funciones.

3. Módulo de perilla (Dial Module)

Este módulo incorpora un control rotatorio que permite ajustes precisos en parámetros como volumen, desplazamiento, brillo o sensibilidad del cursor. Además de la rotación, el dial incluye una acción por presión que habilita funciones secundarias. Su propósito es ofrecer un control táctil rápido y preciso que complemente las funciones estándar de la laptop.

4. Módulo de mini pantalla

Este módulo integra una pantalla de dimensiones reducidas diseñada para mostrar animaciones simples, GIFs, indicadores del sistema, iconografía de estado o retroalimentación visual relacionada con otros módulos del Parma-pad. Su finalidad es añadir un componente visual altamente personalizable que pueda adaptarse a los flujos de trabajo del usuario o a estética temática.

Objetivo del proyecto

Parma-pad busca crear un ecosistema de interacción modular que amplíe el alcance físico de una laptop sin afectar su portabilidad. El enfoque principal está en:

Mantener compatibilidad universal con distintos modelos de laptops mediante un sistema de fijación adaptable.

Permitir que el usuario seleccione únicamente los módulos que necesite.

Integrar controles físicos adicionales sin depender de dispositivos externos grandes.

Ofrecer una estética coherente y robusta inspirada en diseños orientados al gaming y al alto rendimiento.

Facilitar la personalización tanto en funciones como en disposición física.

Alcance actual y futuro

El repositorio documentará todas las fases del proyecto, desde los conceptos iniciales y prototipos de diseño hasta los modelos finales, esquemas de PCB, pruebas mecánicas y software de configuración. En fases futuras se contemplan:

Nuevos módulos especializados.

Versiones alternativas del rail base para distintos formatos de laptop.

Mejoras en la experiencia de configuración y personalización.

Optimización de las carcasas y mecanismos de acoplamiento.
